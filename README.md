1️⃣ What is the difference between var, let, and const?

Answer: Difference between var, let, and const are written below:
var can be redeclared and is function-scoped,
let cannot be redeclared and is block-scoped, and
const is block-scoped and its value cannot be reassigned.

2️⃣ What is the spread operator (...)?
Answer: The spread operator (...) is used to expand elements of an array or object into another array or object.

3️⃣ What is the difference between map(), filter(), and forEach()?
Answer: Difference between map(), filter(), and forEach() are written below :
map() creates a new array after transforming each element,
filter() creates a new array with elements that pass a condition, and
forEach() only loops through the array without returning a new array.

4️⃣ What is an arrow function?
Answer: An arrow function is a shorter syntax for writing functions in JavaScript using =>.

5️⃣ What are template literals?
Answer: Template literals allow programmer to create strings using backticks ( ) and insert variables using ${}.
